Two Types of Components:Functional and ClassBased components _____________________________________________________________________________________ A class component must include the extends React.Component statement. This statement creates an inheritance to React.Component, and gives your component access to React.Component's functions. The component also requires a render() method, this method returns HTML. _____________________________________ app.js: import React from 'react'; class ClassComp extends React.Component { render() { return (
